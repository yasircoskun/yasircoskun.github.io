İsteyen başlıklara tıklayarak web araması yapabilir.

## [GraphQL](https://duckduckgo.com/?q=GraphQL)

Graph Query Language'ın kısaltması. Graf veri yapısı sayesinde varlık ilişkilerini modelleyerek hızlı ve basit sorgulama imkanı sağlayan yeni nesil sorgulama dili.

## [NodeJS](https://duckduckgo.com/?q=NodeJS)

NodeJS, javascript kodlarının backend tarafında çalışan hali olarak düşünülebilir. NodeJs ile console, web, mobil uygulama geliştirilebilir. Bir framework değil programlama dilidir.

## [Express.js](https://duckduckgo.com/?q=Express.js)

NodeJS için web sunucusu.

## [Apollo Server](https://duckduckgo.com/?q=Apollo+Server)

GraphQL için NodeJS paketi.
ExpressJS üzerinde çalışır.
Web arayüzü (swagger) gibi api testi yapmakta ve api projesini belgelemekte faydalı olabilir.
